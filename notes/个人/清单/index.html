    【autoHotkey-changeKeys.ahk】：

     Esc 映射成 AltTab
*Esc::send "{Blind}!{Tab}"

; 左Alt ->Ctrl
*LAlt::LCtrl

^1::Home
^2::End

; 左Ctrl->Enter
*LCtrl::Enter

; Capslock键->BackSpace键
*Capslock::BackSpace

;大写键->回车键
*MButton::Enter

; `键 -> Tab键
*`::Tab

;win+鼠标滚轮上(下滚)滚增加(减少)音量
A_HotkeyInterval := 0
#WheelUp::Volume_Up
#WheelDown::Volume_Down

;win+点击鼠标滚->ESC
#MButton::ESC

;禁用按win键开始菜单弹出,快捷键不受影响
~LWin::Send "{Blind}{vkE8}"

; `2键 -> .键
*F2::.
; `3键 -> /键
*F3::/
; `4键 -> ;键
*F4::;
; `F5键 -> '键
*F5::'
