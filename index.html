介绍

这是一个用于记录技术文档、个人心得、个人收藏的笔记本，创立于 23 年 7 月。 笔记本的名字叫做 流笔， 意为流动不止的笔，即希望多动笔不要让笔停下来。

winget install Hugo.Hugo

运行hugo 

hugo server -p 80 --navigateToChanged  --disableFastRender
